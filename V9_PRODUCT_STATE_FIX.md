# VECTRA v9 — Product State Fix

Внедрено по финальному ТЗ:

1. `все` — режим отображения полного списка, без API-пересчета.
2. Полный список (`all_block`) становится выбираемым через `last_list_items`.
3. `назад` из режимов `all/reasons` возвращает цифры текущего объекта, а не делает pop stack.
4. `назад` после drilldown остается переходом вверх через stack.
5. Убрана отдельная команда/подсказка `искать` из UI.
6. Убраны упоминания `возврат к цифрам`.
7. Цель для не-business объектов выравнивается по дренажу объекта.
8. SKU получает решение по цене: фокус цены, ориентир по цене, действие.
9. Причины отрисовываются читабельно, многострочным блоком.
10. Плюсы/минусы в списках форматируются явно.

Запреты сохранены:
- GPT/UI не считает цифры.
- Цифры приходят из API/backend.
- GPT/UI только отображает и интерпретирует готовые блоки.
