# VECTRA — Release Manager v2.0 + Development Journal v2.0

Deploy package for VECTRA Release Manager v2.0 and Development Journal v2.0 stabilization.

## Main changes

- Release Brief receipt can directly start Product Acceptance through `/release-manager/accept-release-brief`.
- Release Manager records every completed acceptance check in Development Journal, even if no defects were found.
- Product Owner Report now separates current-release quality from previously open engineering tasks.
- Development Journal remains the single source for Laboratory.
- Product Owner receives human-readable results, not engineering diagnostics.

## Artifacts

This release is distributed with:

1. Deploy ZIP.
2. Release Brief.
3. Updated Instruction only if changed.

Instruction was not changed in this release.
