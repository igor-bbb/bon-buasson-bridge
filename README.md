# VECTRA DOP-0003 Deploy Package

Professional Responsibility Lifecycle Engine.

This deploy package contains the VECTRA API with Digital Organization Protocol modules:

- DOP-0001 Document Contract Model
- DOP-0002 Responsibility Transfer Protocol
- DOP-0003 Professional Responsibility Lifecycle Engine

DOP-0003 treats professional responsibility as the central lifecycle object. Documents are carriers of professional state, not the main object of governance.
